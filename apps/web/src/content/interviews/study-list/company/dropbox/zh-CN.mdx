Dropbox 在其前端工程面试环节中更侧重于实际问题，而不是原始的算法编码。

> 我们的 Web 开发人员面试侧重于 HTML/CSS/JS 在构建 Web UI 方面的实际应用。 这些问题基于真实的例子，而不是原始的编码练习。 你将被提供一个设计规范，并被要求构建你能展示你的经验和对 Web 开发技术掌握的最佳解决方案。
