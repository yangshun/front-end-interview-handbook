由于是谷歌，候选人可以期望被测试基础计算机科学概念以及他们的前端知识/技能。

> Web 前端：您应该准备好涵盖前端延迟和使用惯用 JavaScript 实现标准 CS 算法等主题。 您应该能够阐明 JavaScript 的优点和缺点，并准备好涵盖以下任何内容：Web 安全问题 (XSS, XSRF)、原型继承、DOM API 和操作、CSS 操作、浏览器/DOM 事件和事件处理、XHR 请求和 HTTP 标头、JavaScript 闭包
