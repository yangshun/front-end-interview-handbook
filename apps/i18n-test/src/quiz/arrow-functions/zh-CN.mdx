## TL;DR

箭头函数提供了一种简洁的语法来在 JavaScript 中编写函数。它们特别适用于在方法和回调中维护 `this` 上下文。例如，在事件处理程序或 `map` 等数组方法中，箭头函数可以简化代码并避免 `this` 绑定问题。

```js live
const numbers = [1, 2, 3];
const doubled = numbers.map((n) => n * 2);
console.log(doubled); // [2, 4, 6]
```

class Button {
  constructor() {
    this.count = 0;
    this.button = document.createElement('button');
    this.button.innerText = '点击我';
    this.button.addEventListener('click', () => {
      this.count++;
      console.log('计数：', this.count);
    });

  }
}
