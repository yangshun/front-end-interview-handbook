实现一个函数 `union(array)`，该函数从所有给定数组中创建一个按顺序排列的唯一值数组。

```js
union(arrays);
```

## 参数

1. `[arrays]` *(...Array)*: 要检查的数组

## 返回值

*`(Array)`*: 返回合并值的新数组。

## 例子

```js
union([4, 2, 8, 6]); // => [4, 2, 8, 6]

union([2], [1, 2]); // => [2, 1]

union([1, 2, 2]); // => [1, 2]
```

如果 `array` 为空，则该函数应返回一个空数组。

```js
unionBy([]); // => []
```
