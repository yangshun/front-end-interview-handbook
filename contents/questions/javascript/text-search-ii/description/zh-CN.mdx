---
title: 文本搜索 II
excerpt: 实现一个函数，如果搜索词出现在文本中，则突出显示文本
---

在浏览器中，我们可以通过使用 <kbd>Ctrl</kbd> + <kbd>F</kbd> (Windows, Linux) 或 <kbd>⌘</kbd> + <kbd>F</kbd> (Mac) 并输入搜索词来查找网页中的特定单词或短语。 出现的匹配项将以黄色突出显示。

让我们实现一个简单版本的浏览器内网页搜索，不同之处在于我们给定一个字符串（而不是 HTML），搜索匹配项以 **粗体** 显示。

给定一个字符串和一个查询数组，实现一个函数 `textSearch`，该函数从字符串中的 `queries` 数组中查找所有不区分大小写的匹配项，并将匹配项包装在 `<b>...</b>` 标签中。

## 例子

```js
textSearch('The Quick Brown Fox Jumps Over The Lazy Dog', ['fox']);
// 'The Quick Brown <b>Fox</b> Jumps Over The Lazy Dog'
textSearch('The quick brown fox jumps over the lazy dog', ['fox', 'dog']);
// 'The quick brown <b>fox</b> jumps over the lazy <b>dog</b>'
```

如果两个这样的查询重叠或连续，则它们应该被包装在一对 `<b>` 标签中。

```js
textSearch('This is Uncopyrightable!', ['copy', 'right']);
// 'This is Un<b>copyright</b>able!'
textSearch('This is Uncopyrightable!', ['copy', 'right', 'table']);
// 'This is Un<b>copyrightable</b>!'
```

一个字符不会多次匹配同一个查询，较早的字母优先。

```js
textSearch('aaa', ['aa']);
// '<b>aa</b>a'
// 这是因为第二个字符不能再次用作匹配项。
textSearch('aaaa', ['aa']);
// '<b>aaaa</b>'
```

您可以假设 `queries` 数组中没有重复的字符串。
