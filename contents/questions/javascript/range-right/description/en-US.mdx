---
title: Range Right
excerpt: Implement a function that returns a sequence of numbers in descending order
---

Implement a function `rangeRight([start=0], end, [step=1])` that creates an array of numbers progressing from `start` up to (but not including) `end` with a specified `step`, similar to [`range`](/questions/javascript/range), but in a descending order.

## Arguments

1. `start` _(Number)_: The start of the range.
1. `end` _(Number)_: The end of the range.
1. `step` _(Number)_: The value to increment or decrement by.

## Returns

_`(Array)`_: Returns the range of numbers.

## Examples

```js
rangeRight(4); // => [3, 2, 1, 0]

rangeRight(-4); // => [-3, -2, -1, 0]

rangeRight(1, 5); // => [4, 3, 2, 1]

rangeRight(0, 20, 5); // => [15, 10, 5, 0]

rangeRight(0, -4, -1); // => [-3, -2, -1, 0]

rangeRight(1, 4, 0); // => [1, 1, 1]
```

The function should return an empty array if `start` is equal to `end`.

```js
rangeRight(0); // => []
```
