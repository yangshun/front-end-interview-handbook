---
title: 范围
excerpt: 实现一个函数，该函数以升序返回一系列数字
---

实现一个函数 `range([start=0], end, [step=1])`，该函数创建一个数字数组（正数和/或负数），从 `start` 递增到但不包括 `end`。如果指定了负的 `start` 且没有 `end` 或 `step`，则使用 `-1` 作为步长。如果未指定 `end`，则将其设置为 `start`，然后将 `start` 设置为 `0`。

## 参数

1. `start` *(Number)*: 范围的开始。
2. `end` *(Number)*: 范围的结束。
3. `step` *(Number)*: 递增或递减的值。

## 返回值

*`(Array)`*: 返回数字的范围。

## 例子

```js
range(4); // => [0, 1, 2, 3]

range(-4); // => [0, -1, -2, -3]

range(1, 5); // => [1, 2, 3, 4]

range(0, 20, 5); // => [0, 5, 10, 15]

range(0, -4, -1); // => [0, -1, -2, -3]

range(1, 4, 0); // => [1, 1, 1]
```

如果 `start` 等于 `end`，则该函数应返回一个空数组。

```js
range(0); // => []
```
