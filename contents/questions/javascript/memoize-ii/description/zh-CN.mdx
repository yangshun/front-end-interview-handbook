---
title: Memoize II
excerpt: 实现一个函数，该函数返回一个函数的 memoized 版本，该函数接受任意数量的参数
---

memoize 函数是一个高阶函数，它接受一个函数并返回该函数的 memoized 版本。memoized 函数缓存昂贵函数调用的结果，并在再次收到相同的输入时返回缓存的结果。这可以显着提高涉及复杂处理/显着延迟且反复使用相同参数调用的函数的性能。

实现一个函数 `memoize(func)`，它接受一个函数参数 `func` 并返回该函数的 memoized 版本。您可以假设 `func` 仅接受字符串或数字作为参数。

## 例子

```js
function expensiveMul(a, b) {
  console.log('Computing...');
  return a * b;
}

// 创建该函数的 memoized 版本。
const memoizedExpensiveMul = memoize(expensiveMul);

// 第一次调用（计算并缓存结果）。
console.log(memoizedExpensiveMul(3, 7)); // 输出：Computing... 21

// 第二次调用，使用相同的参数（返回缓存的结果）。
console.log(memoizedExpensiveMul(3, 7)); // 输出：21

// 第三次调用，使用不同的参数（计算并缓存新结果）。
console.log(memoizedExpensiveMul(5, 8)); // 输出：Computing... 40

// 第四次调用，使用与第三次调用相同的参数（返回缓存的结果）。
console.log(memoizedExpensiveMul(5, 8)); // 输出：40
```
