实现一个自定义的 `Array` 函数 `Array.prototype.square()`，它创建一个**新**数组，其中包含调用 `.square()` 方法的数组中每个元素的平方结果。

## 例子

```js
[-2].square(); // [4]
[1, 2, 3, 4].square(); // [1, 4, 9, 16]
```

## 注意事项

* 原始数组不应被修改。
* 假设数组只包含数字。
* 假设数组不会是稀疏的，例如 `[1, 2, , 4]`。
