构建一个为指定日期预订单程或往返航班的组件。

## 要求

* 用户可以从两个航班选项中选择："单程航班"和"往返航班"。
* 输入日期字段
  * 日期输入字段表示出发日期和返回日期。
  * 如果选择了“往返航班”选项，则显示返回日期输入，否则隐藏。
* 提交时应针对无效字段进行表单验证：
  * 日期已过。
  * 返回日期早于出发日期。
* 成功提交后，将显示一条消息，通知用户所选内容：
  * 单程航班：“您已预订YYYY-MM-DD的单程航班”
  * 往返航班“您已预订往返航班，出发日期为YYYY-MM-DD，返回日期为YYYY-MM-DD”

对于日期输入字段，[`<input type="date">`元素](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/input/date)将有所帮助。
