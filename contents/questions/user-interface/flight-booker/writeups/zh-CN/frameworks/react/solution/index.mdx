使用不断跟踪输入值的表单（受控表单）比原生表单方法（非受控表单）更方便，因为拥有日期值的状态允许我们将当前出发日期指定为返回日期`<input>`字段的`min`，并在提交期间利用浏览器的表单验证。否则，我们必须在提交处理程序中编写自定义验证逻辑。

在 React 中，受控表单是通过在`<input>`的`onChange`回调中调用状态设置器来实现的。这使得我们可以直接在组件的状态中捕获这些元素的当前值，并且任何状态更改都会立即在 UI 中更新。
