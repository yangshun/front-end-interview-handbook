使用绑定到组件状态的表单比原生表单方法更方便，因为拥有日期值的状态允许我们将当前出发日期指定为返回日期`<input>`字段的`min`，并在提交期间利用浏览器的表单验证。否则，我们必须在提交处理程序中编写自定义验证逻辑。

Vue 中的`v-model`指令使得捕获这些`<input>`元素的当前值并将其与组件状态同步变得简单。
