使用持续跟踪输入值的表单比原生表单方法更方便，因为拥有日期值的状态/属性允许我们将当前出发日期指定为回程日期`<input>`字段的`min`，并在提交期间利用浏览器的表单验证。否则，我们必须在提交处理程序中编写自定义验证逻辑。

使用 Angular 的 `ngModel` 将日期`<input>`和`<select>`元素与组件的状态关联起来。这使得直接将这些元素的当前值捕获为组件的属性变得简单，并且任何属性更改都会立即在 UI 中更新。
