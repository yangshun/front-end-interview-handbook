使用不断跟踪输入值的表单比原生表单方法更方便，因为拥有日期值状态允许我们将当前出发日期指定为回程日期`<input>`字段的`min`，并在提交期间利用浏览器的表单验证。否则，我们必须在提交处理程序中编写自定义验证逻辑。

使用 Svelte 的 `bind:value` 将日期`<input>`和`<select>`元素与组件的状态关联起来。这使得我们可以直接在组件的状态中捕获这些元素的当前值，并且任何状态更改都会立即在 UI 中更新。
