## 解决方案

对于大多数数据获取和显示问题，解决方案可以分解为：

1. 获取数据。
2. 处理数据（如果需要）。
3. 显示数据。

### 获取数据

API 端点以文本格式返回年份列表：

```
1996
1966
1973
1975
1985
2006
2004
...
```

它以单个字符串的形式返回，因此我们需要将数据拆分成一个数组，然后将它们转换为数字。

### 处理数据

要求是显示按十年分组的出生年份直方图。现在我们有了一个数字列表，我们需要对它们进行一些处理，将年份值转换为它对应的桶。例如 1955 -> 1950, 1967 -> 1960。

这可以通过以下方式完成：

1. 除以 10，用 `Math.floor()` 移除小数点，然后再次乘以 10。
2. 模 10 并移除结果。

这两种方法都会正确计算年份的十年。

这是针对每个年份值完成的，我们可以使用一个对象来统计每个十年桶的计数。

### 显示数据

这可能是这个问题中最棘手的部分，因为我们必须绘制与它应该表示的值成比例的条形图。值最大的桶将是最高的条形图。

我们可以将直方图分成三个矩形：

* Y 轴：这呈现每个桶的计数。由于有 200 个数据点分布在 7 个十年中，每个桶平均有 30 多个点，并且 10 个点的比例将提供足够有用的细节。最大 y 轴值是最大桶计数值的四舍五入到最接近的 10 的值。
* X 轴：这呈现十年标签，间隔开来。
* 数据系列：这呈现垂直条，间隔开来。我们还需要分别绘制 y 轴和 x 轴上的线条的左边框和底边框。垂直条的大小使用最大 y 轴值的百分比高度。

请注意，x 轴和数据系列之间的空间应该一致，以便 x 轴的标签与数据条直接对齐。

### 竞态条件

与所有数据获取问题一样，请注意处理竞态条件，并在请求失败时处理错误。
