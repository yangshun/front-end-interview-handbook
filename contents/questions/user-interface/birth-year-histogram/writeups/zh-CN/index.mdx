构建一个小部件，从 [API](https://www.random.org/integers/?num=200\&min=1950\&max=2019\&col=1\&base=10\&format=plain) 获取出生年份数据，并在直方图中显示数据，按十年分组。

## 要求

* 我们将使用 [random.org 的随机数生成器](https://www.random.org/integers/?num=200\&min=1950\&max=2019\&col=1\&base=10\&format=plain) 来模拟一个 API，该 API 返回 1950 年至 2019 年的 200 个随机数据点。
* 通过按十年对数据进行分组，并绘制一个直方图，其中 x 轴是十年分组，y 轴是计数，从而对数据进行制表。
* y 轴应适当标记。

下面的示例显示了一个潜在的最终直方图，但您可以自由地设置其样式。

![出生年份直方图示例](/img/questions/birth-year-histogram/birth-year-histogram-example.png)

## 笔记

* 在实践中，通常不需要对 API 响应数据进行制表。 但是，在面试期间，这是一项可能的任务，因为编写用于制表数据的代码的行为可以很好地表明编码基础知识。
