## 解决方案

由于只需要一个状态值（当前活动的选项卡项），因此在 Vue 中实现一个基本的（并非完全可访问的）Tabs 组件非常简单。Vue 也有助于保持状态和 UI 的同步，这在 Vanilla JavaScript 中做起来比较麻烦。

### Props（API 设计）

构建组件的复杂性部分在于设计其 API。对于 Vue，它将是组件的 props。至少，我们需要以下 props：

* `items`：项目对象列表。每个项目都是一个具有以下字段的对象：
  * `value`：选项卡项目的唯一标识符。
  * `label`：在选项卡项目中显示的文本标签。
  * `panel`：当项目处于活动状态时，在选项卡面板中显示的内容。
* `defaultValue`：要显示的默认选项卡项目/面板。如果未提供 `defaultValue`，我们将使用第一个项目作为值。这是假设 `items` 不为空。
