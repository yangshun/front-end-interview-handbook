构建一个选项卡组件，该组件一次显示一个内容面板，具体取决于活动的选项卡元素。 提供了一些 HTML 作为示例内容。

## 要求

* 单击选项卡使其成为活动选项卡。 为活动选项卡添加视觉指示（例如，使用蓝色文本颜色），以将其与非活动选项卡区分开来。
* 始终只应显示一个面板的内容 - 对应于活动选项卡的内容。

## 笔记

* 这个问题关注的是功能，而不是样式。 除了突出显示活动选项卡之外，无需编写任何自定义 CSS。
* 您可以修改标记（例如，添加 `id`、数据属性、替换某些标签等）并使用客户端渲染。
* 您可能需要考虑改进应用程序用户体验的方法并实施它们（在面试期间这样做可以获得额外加分）。
