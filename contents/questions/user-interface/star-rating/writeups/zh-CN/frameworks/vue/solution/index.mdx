## 解决方案

与 Vanilla 版本相比，使用 Vue 的解决方案要短得多。

该组件将需要一个状态 `hoveredIndex`，用于跟踪当前悬停的星星的索引。

为了确定星星是否应该被高亮/填充，我们检查是否有任何星星当前被悬停，如果是，则优先考虑，任何 `index <= hoveredIndex` 的星星都应该被高亮。 否则，任何 `index < value` 的星星都可以被高亮/填充。
