构建一个购物清单应用程序，您可以在其中搜索杂货，将它们添加到列表中，将项目标记为已选中，并删除已添加的项目。

![购物清单示例](/img/questions/grocery-list/grocery-list-example.png)

## 要求

* 添加项目
  * 用户可以通过在输入字段中键入来添加项目。
  * 输入字段根据用户的输入显示自动完成建议。
  * 可以通过按 Enter 键（这将添加输入中的任何内容）或在建议列表中选择一个项目来添加项目。
  * 添加的项目将出现在下面的未选中状态列表中。
  * 可以多次添加相同的项目。
  * 添加后，输入字段将被清除。
* 将项目标记为已选中
  * 按项目名称旁边的 ⚪ 将其标记为已选中。
  * 选中的项目将变为灰色，⚪ 将替换为 ✅，并且项目名称上会有一条横线。
  * 按 ✅ 将使项目变回未选中状态。
* 删除项目
  * 可以通过按右侧的 ❌ 来删除项目。

## API

### 项目建议

获取杂货建议列表

* URL: `https://api.frontendeval.com/fake/food/:query`
* HTTP 方法: `GET`
* 内容类型: `json`

https://api.frontendeval.com/fake/food/mi 的示例响应：

```js
['milk', 'milkshake', 'mint', 'mixed herbs'];
```

注意：只有两个或更多字符的 `query` 字符串才会返回任何结果。

## 解决方案作者的注意事项（完成后删除）：

* 确保对搜索进行去抖动。
* 将 `<input>` 包装在 `<form>` 中以实现提交到 Enter 行为。
* 不需要为自动完成添加键盘导航（保留）。
* "Check" 和 "Delete" 应该使用 `<button>` 构建，并具有适当的 `aria-label`，例如“将 \[item] 标记为已完成/未完成”和“删除 \[item]”。
