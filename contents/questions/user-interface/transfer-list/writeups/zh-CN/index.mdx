构建一个允许在两个列表之间转移项目的组件。

![Transfer List Example](/img/questions/transfer-list/transfer-list-example.png)

## 要求

* 最初有两个列表，每个列表包含 4 个项目。
* 每个项目都有一个可以选中/取消选中的复选框。
* 转移
  * 单击双箭头按钮将把所有项目从一个列表转移到另一个列表，具体取决于箭头的方向。
  * 单击单箭头按钮将仅转移选定的项目，具体取决于箭头的方向。
  * 转移的项目将添加到目标列表的底部。
  * 项目选择（选中）状态在转移后保留。
  * 如果没有要转移的相关项目，则禁用按钮。
