## 测试用例

1. 对话框打开
   * 验证焦点在对话框打开时移动到对话框中第一个可聚焦的元素
   * 确保焦点被限制在对话框内
   * 检查对话框外的元素不可聚焦
2. 焦点顺序
   * 确认 <kbd>Tab</kbd> 键按逻辑顺序移动焦点通过元素
   * 验证 <kbd>Shift</kbd> + <kbd>Tab</kbd> 反转焦点顺序
   * 确保对话框内的所有交互元素都可聚焦
3. 焦点捕获
   * 测试当使用 Tab 键时，焦点从最后一个元素环绕到第一个元素
   * 验证当使用 <kbd>Shift</kbd> + <kbd>Tab</kbd> 时，焦点从第一个元素环绕到最后一个元素
4. 对话框关闭
   * 确认按下 <kbd>Esc</kbd> 键关闭对话框
   * 验证在对话框外点击会关闭它
   * 确保焦点返回到打开对话框的元素
