title: 注册表单
excerpt: 构建一个注册表单，该表单对用户详细信息进行验证并提交到后端 API
---

构建强大的表单是前端开发人员应具备的核心技能之一。人们每天遇到的最常见的表单之一是注册表单。

实现一个注册表单，允许用户填写他们的详细信息并提交表单。

## 要求

该表单应包含以下具有以下条件的元素：

* 用户名字段
  * 至少 4 个字符。
  * 仅限字母数字。
* 电子邮件字段
  * 有效的电子邮件格式。合理的验证就足够了，您不必严格遵循任何规范。
* 密码字段
  * 至少 6 个字符。
* 确认密码字段
  * 必须与密码字段匹配。
* 提交按钮
  * 包含文本“注册”。
  * 单击提交按钮提交表单。

您可以自由决定何时（在输入/失去焦点/提交时）以及如何（原生 HTML 验证或自定义验证）验证表单。如果验证失败，请在相应的 `<input>` 字段附近（旁边或下方）显示相关错误。

## 提交 API

在 `index.js` 中，已为您实现了一个 `submitForm` 函数，该函数向服务器端 API 发出 AJAX `POST` 请求，该 API 使用相同的条件验证字段。您可以使用它来验证您的表单不允许无效输入。
