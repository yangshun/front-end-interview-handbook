## 测试用例

1. 输入
   * 初始加载时，第一个输入框获得焦点
   * 输入有效输入应填充输入字段并将焦点移动到下一个输入字段
   * 输入字段中不应允许无效输入
   * 网络请求进行时，不应允许任何交互
2. 删除输入
   * 按<kbd>Backspace</kbd>应删除当前输入（如果已填写）或移动到上一个输入字段并删除上一个输入字段中存在的任何输入
3. 导航输入字段
   * 按<kbd>Tab</kbd> / <kbd>Shift</kbd> + <kbd>Tab</kbd>和左右箭头按钮应根据按下的键将焦点移动到左/右
   * 导航后，输入字段应能够被新数字替换，无需任何其他按键
4. 粘贴输入
   * 如果粘贴的代码是有效输入，则粘贴代码
5. “重置”按钮
   * 如果没有输入，则应禁用
   * 单击它应擦除所有输入字段并聚焦于第一个输入
6. “提交”按钮
   * 仅当所有输入字段都填入有效输入时，才应启用提交
