构建一个显示一系列图片的图片轮播。

## 要求

* 图片轮播组件接收一个图片 URL 数组。示例图片 URL 在骨架代码中提供。
* 布局和定位：
  * 图片轮播应居中显示在屏幕上，最大尺寸为 600px * 400px。
  * 图片应缩小以适应轮播，以便整个图片可见。轮播的空白部分可以用黑色填充。
  * 如果屏幕宽度小于图片，则应调整轮播大小以适应可用的水平空间。
* 导航：
  * 添加左右导航按钮，允许用户浏览图片。按钮应允许循环行为，即在最后一张图片之后，图片循环回到第一张。
  * 在底部添加页面按钮以直接跳转到图片。您可以假设图片少于 10 张。
* 此问题不需要动画和过渡，它们将在 [图片轮播 II](/questions/user-interface/image-carousel-ii) 和 [图片轮播 III](/questions/user-interface/image-carousel-iii) 中探讨。

对于这个问题，一个技术限制是任何时候 DOM 中应该只有一个 image 元素。
