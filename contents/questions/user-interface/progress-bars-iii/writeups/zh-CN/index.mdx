**注意：**这是[进度条 II](/questions/user-interface/progress-bars-ii)的高级版本，在尝试此问题之前，您应该先完成该问题。

在[进度条 II](/questions/user-interface/progress-bars-ii)中，我们构建了按顺序填充的进度条，一次一个。 在这个问题中，我们将构建多个同时填充的进度条，最多 3 个。 第四个进度条只有在第一个进度条完全填满后才开始填充。

## 要求

* 单击“添加”按钮会将进度条添加到页面。
* 进度条并行逐渐填充，最多同时填充 3 个。 即，第四个进度条只有在第一个进度条完全填满后才开始填充。
* 每个条大约需要 2000 毫秒才能完全填满。

**提示：**意识到将并发限制更改为 1 会将此问题简化为更简单的[进度条 II](/questions/user-interface/progress-bars-ii)问题。 此问题的解决方案只需要对[进度条 II](/questions/user-interface/progress-bars-ii/solution)的解决方案进行一行更改。
