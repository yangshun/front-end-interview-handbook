**注意：**这是[进度条](/questions/user-interface/progress-bars)的高级版本，在尝试此问题之前，您应该先完成该问题。

构建一个应用程序，点击“添加”按钮将进度条添加到页面。进度条按顺序填充，即第二个进度条只有在第一个进度条完全填充后才开始填充。

## 要求

* 点击“添加”按钮将进度条添加到页面。
* 进度条按顺序逐渐填充，一次一个。即第二个进度条只有在第一个进度条完全填充后才开始填充。
* 每个条大约需要 2000 毫秒才能完全填充。
