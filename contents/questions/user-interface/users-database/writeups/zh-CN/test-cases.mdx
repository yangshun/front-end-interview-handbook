## 测试用例

* 搜索输入
  * 当用户输入时，值会更新
  * 用户列表会根据搜索输入的变化而更新
* 列表框
  * 一次只能选择一个用户
  * 根据搜索输入的变化，显示数据库中所有用户的正确过滤结果
* 名字和姓氏输入
  * 名字和姓氏输入默认为空，并在用户输入时更新
  * 名字和姓氏是创建/更新时提交表单的必填字段
* 表单
  * 当名字或姓氏为空时，创建按钮被禁用。当点击创建按钮时，新用户被添加到列表中
  * 当名字或姓氏为空或未选择用户时，更新按钮被禁用。当点击更新按钮时，选定的用户被更新
  * 当未选择用户时，删除按钮被禁用。当点击删除按钮时，选定的用户被删除。
  * 当未选择用户时，取消按钮被禁用。当点击取消按钮时，选定的用户被清除，名字和姓氏输入为空。
