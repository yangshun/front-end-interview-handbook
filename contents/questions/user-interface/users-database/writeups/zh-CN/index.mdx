构建一个UI，显示数据库（客户端）中的用户列表，并包含以下元素和行为：

## 要求

* 列表框显示数据库中所有用户的列表。一次最多只能选择一个条目。
* 通过在“搜索”输入框中输入字符串，将过滤包含该字符串的用户。过滤立即发生，无需提交/按Enter键。
* “创建”按钮：
  * 仅当未选择任何用户且两个名称字段都已填写时启用。
  * 激活后，将根据名称输入字段创建一个新用户并将其添加到数据库中。然后清除输入字段。
* “更新”按钮：
  * 仅当选择了用户时启用。
  * 激活后，将更新数据库中选定用户的姓名。
* “删除”按钮：
  * 仅当选择了用户时启用。
  * 激活后，将从数据库中删除选定的用户。
* “取消”按钮：
  * 仅当选择了用户时启用。
  * 激活后，将清除任何选定的用户和两个名称输入字段。
