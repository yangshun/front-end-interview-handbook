## 测试用例

* 点击“开始”按钮以启动计时器。观察到：
  * 时间不断增加并显示正确的值。
  * “开始”按钮现在显示“停止”。
* 在计时器运行时单击“停止”。观察到：
  * 时间停止增加。
  * “停止”按钮现在显示“开始”。
* 在停止计时器后单击“开始”按钮。观察到：
  * 时间从停止的计时开始增加，并显示正确的值。
  * “开始”按钮现在显示“停止”。
* 单击“重置”按钮，同时：
  * 计时器正在运行。计时器应停止并重置为 0。
  * 计时器已停止/暂停。计时器应重置为 0。
* 重复上述操作，但单击时间而不是“开始”/“停止”按钮。
