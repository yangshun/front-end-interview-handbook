## 测试用例

1. 功能:
   * 打开和关闭:
     * 验证模态框在触发时打开
     * 确保模态框在单击关闭按钮时关闭
     * 测试任何其他用于打开/关闭模态框的方法（如果适用）
   * 内容定制:
     * 测试传递不同的标题字符串
     * 验证正文中的各种内容类型（文本、HTML、React 组件）
     * 确保关闭按钮始终存在
   * 覆盖:
     * 确认半透明背景覆盖层的存在
     * 验证覆盖层覆盖整个视口
2. 布局和定位:
   * 检查模态框在不同屏幕尺寸上是否水平居中
   * 验证在各种屏幕高度上的垂直居中
   * 测试使用不同的模态框内容大小进行居中
