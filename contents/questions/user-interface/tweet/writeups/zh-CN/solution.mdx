## 解决方案

这个问题评估了开发人员使用 HTML 和 CSS 将设计师的模拟图转化为代码的技能。 涉及的 CSS 并不太复杂，基本使用 flexbox 应该足够了。

* Flex 是满足大多数现代布局需求的理想选择。 熟悉它的工作原理。
* `column-gap` 和 `row-gap` 在这里非常有用，因为如果没有元素，它不会渲染间距（推文操作可能没有计数）。
* 可以在容器上使用 `display: flex; justify-content: space-between` 来实现推文操作之间的等间距。
* 我们可以使用 CSS 自定义属性/变量来保持缩略图和图标的 `height`/`width` 值同步，因为它们是正方形的。 预处理器也可以完成这项工作，但现在更喜欢 CSS 变量，因为它是浏览器原生功能。
