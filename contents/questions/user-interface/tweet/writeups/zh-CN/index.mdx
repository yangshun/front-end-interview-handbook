编写HTML和CSS来构建Twitter的tweet UI。

![Tweet demo](/img/questions/tweet/tweet.png)

尽可能严格地遵循给定的规范。模板中为您提供了示例缩略图和图标SVG。

## 要求

[![Tweet specifications](/img/questions/tweet/tweet-specs.png)](/img/questions/tweet/tweet-specs.png)

* Tweet
  * 背景颜色：`#fff`
  * 边框颜色：`#cfd9de`
  * 边框宽度：`1px`
  * 边框半径：`16px`
  * 填充：`12px`
* 缩略图
  * 尺寸：`48px`
  * 完全圆角
* 姓名
  * 颜色：`#000`
  * 字体：`sans-serif`
  * 字号：`15px`
  * 字体粗细：`bold`
* 元数据
  * 颜色：`#71767b`
  * 字体：`sans-serif`
  * 字号：`15px`
  * 使用`&middot;`表示点
* 消息
  * 颜色：`#0f1419`
  * 字体：`sans-serif`
  * 字号：`15px`
* 操作行
  * 最大宽度：`425px`
  * 每个操作之间等间距
* 图标
  * 颜色：`#536471`
  * 尺寸：`16px`
  * 与计数垂直居中
* 操作计数
  * 颜色：`#536471`
  * 字号：`13px`
