---
title: 文件管理器
excerpt: 构建一个文件管理器组件，以树状层级视图的形式浏览文件和目录
---

给定一个文件对象数组，构建一个组件，以树状格式显示它们。

有两种类型的对象——文件和目录：

1. 文件本质上是树的叶节点，它们没有子节点。
2. 目录可以包含其他对象——文件或目录。

您可以假设同一目录中的 ID 和名称是唯一的。

## 要求

* 应该显示目录/文件的名称。
* 目录
  * 目录的内容应以表明它属于该目录的方式显示。 推荐的方法是将内容缩进到右侧。
  * 目录可以展开和折叠。 单击目录应切换其展开/折叠状态。
  * 目录应显示在文件之前。 所有项目都应在其各自的类别中按字母顺序排序。
  * 只要清楚该项目是目录以及它处于展开还是折叠状态，您就可以按照您喜欢的方式设置展开/折叠功能的样式。
  * 目录可以为空。
* 文件
  * 文件不可展开，也不能与之交互。
* 练习的重点是功能，而不是样式。

## 数据格式

```ts
interface FileObject {
  id: number;
  name: string;
  children?: FileObject[];
}
```

示例数据：

```js
const fileData = [
  {
    id: 1,
    name: 'README.md',
  },
  {
    id: 2,
    name: 'Documents',
    children: [
      {
        id: 3,
        name: 'Word.doc',
      },
      {
        id: 4,
        name: 'Powerpoint.ppt',
      },
    ],
  },
];
```
