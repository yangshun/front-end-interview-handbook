## 测试用例

1. 渲染：
   * 验证组件是否无错误地渲染
   * 检查是否正确显示所有文件和目录名称
   * 确保目录显示在文件之前
   * 确认每个目录中的项目按字母顺序排序
2. 目录展开/折叠：
   * 测试通过单击目录来展开目录
   * 测试再次单击已展开的目录来折叠目录
   * 验证展开/折叠指示器是否相应更改
   * 测试展开/折叠嵌套目录
3. 缩进：
   * 检查目录内容是否正确缩进
   * 验证嵌套目录是否具有适当的缩进级别
4. 文件交互：
   * 确认文件不可展开或交互
5. 空目录：
   * 测试空目录的渲染
   * 验证空目录是否仍可展开/折叠
6. 排序：
   * 确认目录出现在每个级别的文件之前
   * 验证目录和文件内的字母排序
7. 状态持久性：
   * 检查目录的展开/折叠状态在重新渲染时是否保留
