## 解决方案

这个问题分为两个部分：(1) 接受用户输入和 (2) 生成表格。

### 接受用户输入

由于表格应该仅在按下“提交”按钮时创建，因此更新是单向的。一个简单的`<form>`，其中输入值完全存储在 DOM 中就足够了。也可以使用状态或数据绑定（取决于 UI 框架）来跟踪输入值，但我们需要更多的状态值，这是不必要的。

通过使用`<input type="number" min={1}>`，我们确保只能输入数字并利用 HTML 验证，以便字段的最小值是 1。这样我们就不必编写自己的验证。

我们可以监听表单的[`SubmitEvent`](https://developer.mozilla.org/en-US/docs/Web/API/SubmitEvent)，从`SubmitEvent`中获取`FormData`，并从`FormData`中提取行和列的值。请记住在提交回调中执行`event.preventDefault()`，否则表单提交将导致页面重新加载。

### 生成表格

生成表格非常简单。请注意，`<tr>`应该在`<tbody>`内，而不是直接在`<table>`内。表格生成中更棘手的部分是每个表格单元格内的数字，这涉及使用数学公式。每列的数字序列取决于该列是偶数列还是奇数列，可以使用以下公式定义它们：

* 偶数：`rows * col + (row + 1)`
* 奇数：`rows * (col + 1) - row`
