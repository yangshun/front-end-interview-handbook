## 测试用例

* 表格
  * 测试表格是否显示以下列：Id、Name、Age、Occupation。
  * 测试表格中的每一行是否代表一个用户。
  * 表格更新以显示下一页的用户。
* 分页
  * 测试分页控件是否允许导航到上一页。
  * 测试分页控件是否允许导航到下一页。
  * 测试分页控件是否显示当前页码和总页数。
  * 测试页面按钮是否被适当禁用。
* 动态表格更新
  * 测试当用户导航到不同页面时，表格是否动态更新。
  * 使用分页控件导航到不同的页面。表格应反映所选页面的用户。
  * 将选择更改为每页 5、10 和 20 个用户。表格应更新以显示每页所选的用户数。
