## 解决方案

创建一个函数，将 DOM 元素变成一个进度条。然后，我们构建必要的内部 DOM 元素，设置适当的样式，并在其中添加一个标签以指示进度百分比。

使用 `Element.style.width` 属性可以按比例填充进度条（0-100 之间的一个数字，包括 0 和 100）。由于该值是动态的，我们不可能事先为其编写类，因此我们必须使用内联样式。

最好公开一种方法来更改进度百分比，因为进度会随着时间而变化。它的用法在底部的进度滑块中显示。
