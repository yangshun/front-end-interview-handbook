## Test cases

- A 10 x 10 grid is rendered.
- Draw a selection box
  - A box is being drawn on mouse down and the box's position and dimensions follow the drag origin and the mouse's current position.
  - Move the cursor around the drag origin, the box should render correctly.
  - Upon releasing the mouse, the selection box disappears.
  - Start drawing from various positions on the page: outside the grid, within the grid, etc.
- Select cells in the grid
  - While drawing the box over the grid, any overlapping cells should be highlighted.
  - Clicking anywhere on the screen causes the highlighted cells to no longer be highlighted.
