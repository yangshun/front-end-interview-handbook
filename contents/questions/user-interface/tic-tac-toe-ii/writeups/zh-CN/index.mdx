**注意：**这是[井字棋](/questions/user-interface/tic-tac-toe)的高级版本，在尝试此问题之前，您应该先完成该问题。

> 井字棋是一款供两名玩家使用的游戏，他们轮流在三乘三的网格中用 X 或 O 标记空格。 成功在水平、垂直或对角线上放置三个标记的玩家获胜。

*来源：[维基百科](https://en.wikipedia.org/wiki/Tic-tac-toe)*

传统上，井字棋使用 3 x 3 的网格。 在这个问题中，构建一个高级版本的井字棋游戏，该游戏在棋盘上有 N x N 个单元格，并且需要在水平、垂直或对角线上有 M 个标记才能获胜。

下图显示了一个示例游戏，其中 N = 5 且 M = 4。

![N x N 井字棋示例](/img/questions/tic-tac-toe-ii/tic-tac-toe-ii-example.png)

## 要求

* 玩家可以轮流在 N x N 棋盘上放置 X 或 O。
* 如果玩家的 M 个标记在水平、垂直或对角线上，则该玩家获胜。
* 在顶部显示当前游戏状态：轮到谁了、获胜者（如果有）或平局。
* 添加一个“重置”按钮，以便随时重新启动游戏。

如果您不熟悉该游戏，请尝试[在 Google 上玩井字棋](https://www.google.com/search?q=tic-tac-toe)。
