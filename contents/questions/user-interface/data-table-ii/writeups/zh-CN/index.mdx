**注意：**这是对[数据表](/questions/user-interface/data-table)问题的后续，在尝试此问题之前，您应该先完成该问题。

在[数据表](/questions/user-interface/data-table)中，我们构建了一个以分页格式显示用户的数据表。数据表通常允许按升序和降序对特定列中的行进行排序。

## 要求

* 允许通过单击表头按特定列排序。数据将按选定列升序（默认）排序。
* 单击已按该列排序的表头将在升序和降序之间切换排序顺序。

## 笔记

骨架代码使用[数据表](/questions/user-interface/data-table)的解决方案，但您可以自由使用自己的解决方案作为起点。
