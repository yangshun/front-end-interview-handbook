---
title: 联系表单
excerpt: 构建一个联系表单，将用户的反馈和联系方式提交到后端 API
---

构建表单是前端的常见任务。在本练习中，我们将构建一个基本的“联系我们”表单，通常在营销网站上供访问者提问或提供反馈。

## 要求

* 表单应包含以下元素：
  * 姓名字段。
  * 电子邮件字段。
  * 消息字段。由于消息可能很长，`<textarea>` 将更合适。
  * 提交按钮
    * 包含文本“发送”。
    * 单击提交按钮提交表单。
* 表单和提交应完全在 HTML 中实现。对于此问题，请勿使用任何 JavaScript 或特定于框架的功能。
* 无需对字段进行任何客户端验证。验证将在服务器端完成。

## 提交 API

提交后，将表单数据 `POST` 到 `https://questions.greatfrontend.com/api/questions/contact-form`，并在**请求正文**中包含以下字段：`name`、`email`、`message`。

如果所有表单字段都正确填写，您将看到一个包含成功消息的警报。恭喜！

## 笔记

对于这个问题，您不需要 JavaScript，重点是 HTML 表单验证和提交。
