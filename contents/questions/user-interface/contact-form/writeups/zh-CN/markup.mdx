## 解决方案

这是一个基本问题，用于评估您对 Web 平台的了解。表单和表单提交完全可以在没有任何 JavaScript 的情况下构建！作为前端工程师，了解平台提供的功能并且不依赖 JavaScript 来完成所有事情非常重要。

我们需要做的第一件事是将字段包装在 `<form>` 中，模板中已经给出了。为了告诉 `<form>` 将数据提交到哪个 URL，我们使用 `action` 属性，并将 API URL 作为值。API URL 期望一个 HTTP POST 请求，因此我们也在 `<form>` 上使用 `method="post"`。

具有 `name` 属性的表单字段将使属性值成为表单数据中的键。因此，我们可以将 `name="name"`、`name="email"`、`name="message"` 添加到各种表单字段中。还要添加 `<label>` 来标记你的 `<input>`。

最后，提交按钮可以通过两种方式实现：

1. `<button type="submit">`：默认情况下，`<button>` 的 `type="submit"` 为默认值，当在 `<form>` 中使用时，将触发表单提交。
2. `<input type="submit">`：该元素将呈现为一个按钮，单击 `<input>` 将触发表单提交。
