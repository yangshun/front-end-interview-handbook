**注意：**这是[进度条 III](/questions/user-interface/progress-bars-iii)的高级版本，在尝试此问题之前，您应该先完成该问题。

在[进度条 III](/questions/user-interface/progress-bars-iii)中，我们构建了并行填充的进度条，其中多个进度条同时填充，最多 3 个。 在这个问题中，我们将允许暂停和恢复动画。

## 要求

* 最初将有一个空的进度条。
* 进度条并行逐渐填充，最多同时填充 3 个。 即第四个进度条只有在第三个进度条完全填满后才会开始填充。
* 每个条大约需要 2000 毫秒才能完全填满。
* 使用以下功能实现以下按钮：
  * 开始/暂停：开始/暂停进度条的填充。 当动画播放时，按钮标签变为“暂停”。
  * 添加：将一个新的进度条附加到列表的底部。
  * 重置：重置为初始状态，其中只有一个空条，并停止任何正在运行的动画。
