## 测试用例

1. 游戏开始
   * 验证当点击“开始”按钮时，回合开始。
   * 检查计时器是否从指定的时间限制开始倒计时。
   * 确保地鼠开始随机出现在洞中。
2. 击打地鼠
   * 测试每隔`molesAppearingInterval`，出现`molesAtOnce`个地鼠。
   * 点击可见的地鼠，验证地鼠消失。
   * 确认成功“击打”地鼠时，分数增加一分。
   * 确保点击不可见的洞不会影响分数。
3. 游戏结束
   * 验证当计时器达到零时，回合结束。
   * 检查游戏结束后，地鼠停止出现。
   * 确保正确显示最终分数。
   * 测试游戏结束后，“再玩一次”按钮出现，点击后开始新一轮游戏。
4. 游戏逻辑
   * 验证地鼠随机地从不同的洞中出现和消失。
   * 通过设置不同的时间限制来测试时间限制功能，并验证游戏是否相应地结束。
   * 验证击中地鼠时分数加一，而未击中地鼠时分数不受影响。
5. 边缘情况
   * 测试当发生意外输入或无效操作时（例如，快速点击多个洞）的游戏行为。
