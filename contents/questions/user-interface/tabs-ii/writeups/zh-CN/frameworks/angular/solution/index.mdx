## 解决方案

我们将在[Tabs' solution](/questions/user-interface/tabs/angular/solution)的基础上构建。除了添加正确的 ARIA 角色和状态（这很简单）之外，我们还需要将选项卡项与相应的选项卡面板链接起来。因此，我们创建了两个函数 `getTabListItemId` 和 `getTabPanelId` 来执行此操作。

* `getTabListItemId` 生成一个唯一的 ID 字符串，用作选项卡项的 `id` 属性的值。此 ID 将用作相应选项卡面板的 `aria-labelledby` 属性的值。
* `getTabPanelId` 生成一个唯一的 ID 字符串，用作选项卡面板的 `id` 属性的值。此 ID 将用作相应选项卡项的 `aria-controls` 属性的值。

由于页面上可能存在多个 `Tabs` 组件实例，并且我们无法保证选项卡项值是全局唯一的，因此每个 `Tabs` 实例都需要一个唯一的标识符。`useId` React hook 可用于为每个 `Tabs` 实例生成唯一的 ID。最终的 ID 字符串将是 `Tabs` 实例的 ID、项目值以及它是选项卡项还是选项卡面板的串联。
