在[数据表 III](/questions/user-interface/data-table-iii)中，我们构建了一个通用数据表，该表以分页格式显示行，并允许按升序和降序对特定列中的行进行排序。`DataTable`组件未与数据耦合，可用于呈现任何类型的数据。

扩展`DataTable`组件以允许按列过滤功能。

## 要求

*  每个列都可以配置为在标题单元格内具有一个过滤器输入
*  过滤器输入过滤与输入值匹配的行
*  支持的过滤器类型：
  *  **文本搜索**：对于字符串字段。基于不区分大小写的字符串比较来匹配数据
  *  **范围搜索**：对于数字字段。为简单起见，使用两个文本字段，指示最小值/最大值。如果该值介于提供的最小值/最大值之间，则匹配
*  如果任何过滤器输入为空或无效，则应忽略它们
*  应仅保留通过**所有**现有过滤条件的行

为了帮助您进行测试，使用房屋数据（`src/data/houses.json`）呈现了另一个表。在处理问题时，可以随意注释掉任何代码。您应该使用这两个数据集测试您的最终解决方案，以确保该组件保持通用性和灵活性。

**提示**：如果您需要有关设计用于添加过滤器的 API 的灵感，请查看此[TanStack Table Column Filters 示例](https://tanstack.com/table/latest/docs/framework/react/examples/filters)。
