## 测试用例

1. 初始显示
   * 打开游戏并验证是否正确显示了面朝下的卡片网格。
   * 检查网格中的卡片数量是否符合预期。
   * 确保所有卡片都面朝下，并且没有显示其图像。
2. 翻牌
   * 点击一张卡片并验证它是否显示其图像。
   * 点击另一张卡片并验证它是否也显示图像。
   * 确保如果选中的卡片匹配，则它们保持正面朝上。
   * 验证如果选中的卡片在短暂延迟后不匹配，则它们会翻转回去。
3. 匹配对
   * 选择两张具有相同图像的卡片，并验证它们是否保持正面朝上。
4. 不匹配对
   * 选择两张具有不同图像的卡片，并验证如果未选择新卡片，它们是否在短暂延迟后翻转回去。
   * 选择两张具有不同图像的卡片，然后选择一张新卡片，前两张选中的卡片应该翻转回去。
5. 游戏完成
   * 玩游戏直到所有对都成功匹配。
   * 验证是否显示了“再玩一次”按钮，并且单击它会重置游戏。
6. 新游戏/重置
   * 开始新游戏或重置当前游戏。
   * 验证网格是否已重置，所有卡片都面朝下。
