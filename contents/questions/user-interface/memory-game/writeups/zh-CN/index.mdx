构建一个记忆游戏，玩家需要匹配成对的卡片。

![Memory game example](/img/questions/memory-game/memory-game-example.png)

## 要求

* 在屏幕上显示一个朝下的卡片网格，每张卡片代表一个不同的项目或图像。您可以使用表情符号作为图像，已提供表情符号列表。
* 网格应包含相等数量的卡片以配对。
* 当玩家点击一张卡片时，它应该翻转并显示其图像。
* 允许玩家一次选择两张卡片。
* 如果所选的两张卡片具有相同的图像，则匹配，它们应保持正面朝上。
* 如果所选的两张卡片具有不同的图像，并且玩家
  * 选择其他卡片，则所选的两张卡片应翻转回去。
  * 什么都不做，两张选定的卡片应在短暂的延迟后翻转回背面。
* 成功匹配所有对后，结束游戏并显示“再玩一次”按钮。
