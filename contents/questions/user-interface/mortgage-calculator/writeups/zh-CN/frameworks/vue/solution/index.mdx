此问题评估您是否了解如何允许用户输入表单输入、验证它们并根据输入执行计算。

## 表单标记

要收集用户数据，我们使用一个`<form>`元素和`<input>`字段。

要验证输入，请使用内置的输入类型验证器（`<input type="number">`），或使用`Number()`构造函数将用户输入转换为数字并检查`NaN`值。您可以在 HTML 中使用`required`属性来确保填写所有输入字段，并在提交事件回调被触发之前使用`min`属性来确保输入字段具有有效的最小值。

我们不需要为表单字段使用受控值，因为我们只需要在提交期间进行计算。

## 表单提交

我们可以使用 JavaScript 将`click`事件附加到计算公式的“计算”按钮上，以计算抵押贷款的还款额。但最好将`submit`事件处理程序添加到`<form>`中，以便单击“计算”按钮和在任何输入字段中按“Enter”键都会提交表单。还需要使用`.prevent`事件修饰符来拦截浏览器提交操作，该操作默认情况下会触发页面重新加载，而此处不需要。

## 显示结果

使用公式计算出 3 个必要的金额后，使用`Intl.NumberFormat()`将金额格式化为 2 位小数，并带有美元符号。随意自定义格式以适应您自己的语言环境和货币。
