这个问题评估您是否了解如何允许用户输入表单输入、验证它们并根据输入执行计算。

## 表单标记

为了收集用户数据，我们使用一个`<form>`元素和`<input>`字段。

要验证输入，请使用内置的输入类型验证器（`<input type="number">`），或使用`Number()`构造函数将用户输入转换为数字并检查`NaN`值。您可以在 HTML 中使用`required`属性来确保所有输入字段都已填写，并在提交事件回调被触发之前使用`min`属性来确保输入字段具有有效的最小值。

## 表单提交

我们可以使用 JavaScript 将点击事件侦听器附加到“计算”按钮，该按钮使用公式计算抵押贷款还款额。但最好将一个`'submit'`事件侦听器添加到`<form>`中，这样点击“计算”按钮和在任何输入字段中点击“Enter”都会提交表单。 同样有必要使用`event.preventDefault()`来拦截浏览器提交操作，该操作默认触发页面重新加载，而此处不需要。

## 显示结果

使用公式计算出 3 个必要的金额后，使用`Intl.NumberFormat()`将金额格式化为 2 位小数，并带有美元符号。 随意自定义格式以适应您自己的语言环境和货币。
