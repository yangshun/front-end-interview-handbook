## 测试用例

* 验证计数器从 0 开始。
* 多次点击“/2”、“-1”、“+1”、“x2”按钮。 观察：
  * 计数器应在执行操作后更新为正确的值。
  * 历史记录表中应在顶部添加一行，显示操作、操作前的计数和操作后的计数。
* 多次点击“撤消”按钮。 观察：
  * 计数器应更新为正确的“旧”值。
  * 历史记录表中将删除顶部行。
* 多次点击“重做”按钮。 观察：
  * 计数器应更新为正确的“新”值。
  * 历史记录表中应重新添加重做行。
* 点击“重置”按钮。 观察：
  * “撤消”和“重做”按钮应被禁用。
  * 历史记录表中不应有任何内容。
  * 计数器应重置为 0。
* 在撤消某些操作后，单击“/2”、“-1”、“+1”、“x2”按钮。
  * 撤消的操作不能再重做。
  * “重做”按钮应被禁用。
* a11y
  * 使用 Tab 键循环切换按钮。
  * 能够使用空格键和回车键与页面交互。
