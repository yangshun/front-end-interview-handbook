构建一个具有更改历史记录的计数器，并能够撤消/重做操作。

![可撤销计数器示例](/img/questions/undoable-counter/undoable-counter-example.png)

## 要求

* 计数器从 0 开始。
* 按钮 "/2", "-1", "+1", "x2" 将相应的操作应用于当前计数。
  * 一行添加到历史记录表中，显示操作、操作前的计数、操作后的计数。
* "撤消" 按钮撤消最后一个操作并更新计数。
* "重做" 按钮应用最后一个被撤消的操作（如果有）。
* 单击修改按钮应清除任何被撤消的操作，并且它们不能被重做。
* "重置" 按钮将计数器重置为 0 并清除所有历史记录。
