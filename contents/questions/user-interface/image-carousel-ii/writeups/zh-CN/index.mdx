**注意：**这是[图片轮播](/questions/user-interface/image-carousel)问题的后续，在尝试此问题之前，您应该先完成该问题。

构建一个显示一系列图像的图像轮播，在图像之间导航时平滑过渡。

## 要求

* 应支持[图片轮播](/questions/user-interface/image-carousel)问题的所有要求。
* 单击“上一个/下一个”按钮时，当前图像应平滑地移出，而下一张图像进入。过渡方向分别为向左/向右。
* 单击页面按钮时的过渡是可选的，但强烈建议。 旨在重用“上一个/下一个”按钮的过渡代码。
* **建议**：将所有图像渲染在容器内的水平行中，并操纵容器的水平偏移量以实现过渡。

单击“查看您正在构建的内容”按钮以查看结果的预览。

## 笔记

框架代码使用[图片轮播](/questions/user-interface/image-carousel)的解决方案，但您可以自由使用自己的解决方案作为起点。
