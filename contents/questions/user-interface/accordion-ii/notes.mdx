## Accessibility

By using a `<button>` for the section titles, we have enabled the basic keyboard interactions necessary to achieve sufficient accessibility for accordion components. However, there are some useful optional keyboard interactions to add, which will be covered in [Accordion III](/questions/user-interface/accordion-iii).
