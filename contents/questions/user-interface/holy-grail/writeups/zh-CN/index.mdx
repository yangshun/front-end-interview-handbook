圣杯布局是一种著名的 CSS 页面布局，传统上很难实现。它由页眉、页脚和三列组成。左侧列包含导航项目，中间列包含页面内容，右侧列包含广告。

![圣杯布局示例](/img/questions/holy-grail/holy-grail-example.png)

仅使用 CSS 实现圣杯布局。您不需要更改太多 HTML。

## 要求

* 页眉
  * 水平延伸到整个页面。
  * 60px 高。
* 列
  * 左右两列的固定宽度均为 100px。
  * 中间列是流体宽度。
  * 所有列的高度都应该相同，无论哪一列最高。
