> 井字棋是一款供两名玩家进行的游戏，他们轮流在一个三乘三的网格中用 X 或 O 标记空格。 成功在水平、垂直或对角线上放置三个标记的玩家获胜。

*来源：[维基百科](https://en.wikipedia.org/wiki/Tic-tac-toe)*

构建一个可供两名玩家玩井字棋游戏。

![井字棋示例](/img/questions/tic-tac-toe/tic-tac-toe-example.png)

## 要求

* 玩家可以轮流在棋盘上放置 X 或 O。
* 如果一名玩家的三个标记位于水平、垂直或对角线上，则该玩家获胜。
* 在顶部显示当前游戏状态：轮到谁了、获胜者（如果有）或平局。
* 添加一个“重置”按钮，以便随时重新开始游戏。

如果您不熟悉这款游戏，请尝试[在 Google 上玩井字棋](https://www.google.com/search?q=tic-tac-toe)。
