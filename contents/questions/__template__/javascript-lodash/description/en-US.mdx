Implement a function `todoReplaceMe(foo)` that returns the double of `foo`.

Obtain description here: https://raw.githubusercontent.com/lodash/lodash/4.17.15/doc/README.md

## Arguments

1. `foo` _(number)_: The number to double.

## Returns

_`(number)`_: Returns the double of value.

## Examples

```js
todoReplaceMe(0); // => 0

todoReplaceMe(1); // => 2

todoReplaceMe(2); // => 4
```
