给定一个字符串 `str`，确定它是否是回文。如果是，则返回 `true`，否则返回 `false`。

如果一个字符串在将所有大写字母更改为小写字母并丢弃所有非字母数字字符后，正向和反向读取时保持相同，则该字符串是回文。字母数字字符由字母和数字组成。

### 输入

- `str: string`: 字符串

### 约束

- 1 &lt;= `str.length` &lt;= 1000
- 字符串 `str` 仅由可打印的 ASCII 字符组成
