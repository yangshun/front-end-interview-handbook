考虑一个具有`num`个节点的图，从`0`到`num - 1`进行标记。给定`num`和边列表，其中每个`edges[i] = [a, b]`表示节点`a`和`b`之间的无向连接，确定图中连通分量的数量。

在图的上下文中，无向图的[component](<https://en.wikipedia.org/wiki/Component_(graph_theory)>)是一个连通的子图，它不是任何更大的连通子图的一部分。

### 输入

- `num: number`: 一个整数，图中节点的数量
- `edges: Array<[number, number]>`: 一个二维数组，其中`edges[i] = [a, b]`表示节点`a`和`b`之间的无向边

<MDXTestExamples testCases={runTestCases} />

### 约束

- 1 &lt;= `num` &lt;= 1000
- 1 &lt;= `edges.length` &lt;= 2000
- `edges[i].length` == 2
- 0 &lt;= `a` \< `b` \< `num`
- `a != b`
- 没有自循环或重复边
