实现一个执行**递归、就地**快速排序的函数。该函数应接收一个整数数组。输出应该是一个按升序排序的数组。

## 例子

```js
quickSort([9, 3, 6, 2, 1, 11]); // [1, 2, 3, 6, 9, 11]
quickSort([12, 16, 14, 1, 2, 3]); // [1, 2, 3, 12, 14, 16]
```

## 回顾

快速排序是一种高效的、就地的、递归的排序算法，它选择一个“枢轴”元素，并将所有其他元素分成 2 个子数组：

1. 小于枢轴的元素被添加到位于枢轴之前的 1 个子数组中。

然后将快速排序递归地应用于每个子数组。一旦子数组被排序，它们就会按照上述方式与它们之间的枢轴元素合并。
