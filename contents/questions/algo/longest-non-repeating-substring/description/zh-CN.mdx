给定一个字符串`str`，确定不包含任何重复字符的最长子串的长度。

一个**子串**是字符串中任意连续的字符序列。例如，字符串`abc`的子串是`a`、`b`、`c`、`ab`、`bc`和`abc`。子串是通过选择一个起始点和一个结束点，并且不跳过中间的字符而形成的。

### 输入

- `str: string`: 字符串

### 约束

- 0 &lt;= `str.length` &lt;= 10,000
- `str` 仅包含小写英文字母
