给定一个字符串`str`，确定作为回文的子串的总数。

一个**回文**是指从前往后和从后往前读都一样的字符序列（例如，`racecar`）。

一个**子串**是字符串中任何连续的字符序列。例如，字符串`abc`的子串是`a`、`b`、`c`、`ab`、`bc`和`abc`。子串是通过选择一个起始点和一个结束点形成的，中间没有跳过字符。

### 输入

- `str: string`: 字符串
