给定一个`intervals`数组，其中每个区间表示为`[start, end]`。每对表示数轴上区间的开始和结束。例如，`[1, 3]`表示一个从1开始到3结束的区间。

找到需要移除的最小区间数，使剩余的区间不相交。

如果数轴上没有两个区间重叠，则认为区间不相交。例如：

- `[1, 3]` 和 `[4, 6]` 不相交，因为它们没有任何共同点
- `[1, 5]` 和 `[4, 6]` 不相交，因为它们在4和5之间重叠

### 输入

- `intervals: Array<[number, number]>`: 整数对数组
