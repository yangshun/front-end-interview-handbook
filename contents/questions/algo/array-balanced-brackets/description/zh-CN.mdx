给定一个字符串 `str`，由诸如 `'(', ')', '{', '}', '[ 和 ']'` 等字符组成，确定输入字符串是否正确平衡。

如果满足以下条件，则认为字符串是平衡的：

- 每个左括号都由相同类型的右括号闭合（例如，`(` 对应 `)`, `{` 对应 `}`, `[` 对应 `]`）
- 括号按正确的顺序闭合（例如，`([])` 有效，但 `([)]` 无效）
- 任何包含在一对匹配括号内的括号子集也必须形成一个有效的匹配对（例如，`{[(])}` 不平衡，因为 `{` 和 `}` 内部的内容不平衡）。

### 输入

- `str: string`: 字符串
