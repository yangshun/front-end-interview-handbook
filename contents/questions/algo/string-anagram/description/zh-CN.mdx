给定两个字符串`str1`和`str2`，确定`str2`是否是`str1`的异位词，如果是则返回`true`，否则返回`false`。

异位词是指通过重新排列另一个单词或短语的字母而形成的单词或短语，使用所有原始字母且仅使用一次。例如，单词`listen`和`silent`是异位词，因为它们使用相同的字母，具有相同的频率，但顺序不同。

### 输入

- `str1`：一个字符串
- `str2`：一个字符串
