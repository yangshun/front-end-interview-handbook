给定一个会议时间 `intervals` 数组，其中每个元素代表一个会议时间，格式为 `[start, end]`，确定一个人是否可以参加所有会议，而会议时间之间没有任何重叠。

### 输入

- 一个整数的二维数组 `intervals`，大小为 `n x 2`，其中 `n` 是会议的数量，每个区间 `[start, end]` 代表一个会议，开始时间为 start，结束时间为 end

<MDXTestExamples testCases={runTestCases} />

### 约束

- 0 &lt;= `intervals.length` &lt;= 1000
- `intervals[i].length` == 2
- 0 &lt;= `start` \< `end` &lt;= 1,000,000
