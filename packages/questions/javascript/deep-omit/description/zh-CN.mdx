实现一个函数 `deepOmit(obj, keys)`，该函数从对象中删除指定的键及其对应的值，包括嵌套对象或数组。它通过递归方式遍历整个对象结构，确保在所有级别删除指定键的所有出现。该函数接受一个对象 (`obj`) 和一个字符串键数组 (`keys`)。

## 例子

```js
deepOmit({ a: 1, b: 2, c: 3 }, ['b']); // { a: 1, c: 3 }
```

一个更复杂的嵌套对象示例：

```js
const obj = {
  a: 1,
  b: 2,
  c: {
    d: 3,
    e: 4,
  },
  f: [5, 6],
};
deepOmit(obj, ['b', 'c', 'e']); // { a: 1, f: [5, 6] }
```
