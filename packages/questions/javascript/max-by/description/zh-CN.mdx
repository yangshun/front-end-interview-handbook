---
title: 最大值依据
excerpt: 实现一个基于指定条件查找最大元素的函数
---

实现一个函数 `maxBy(array, iteratee)`，该函数在遍历 `array` 后，根据 `iteratee` 的结果找到最大值的元素。`iteratee` 会传入一个参数：(value)。

## 参数

1. `array` *(Array)*: 要迭代的数组。
2. `iteratee` *(Function)*: 每次迭代调用的函数。

## 返回值

*`(*)`*: 返回最大值。

## 例子

```js
maxBy([{ n: 1 }, { n: 2 }], (o) => o.n); // => { n: 2 }

maxBy([1, 2], (o) => -o); // => 1
```

该函数应该忽略 `iteratee` 产生 `null` 或 `undefined` 的元素。

```js
maxBy([{ n: 1 }, { n: 2 }], (o) => o.m); // => undefined
```
