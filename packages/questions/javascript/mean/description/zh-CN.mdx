title: 均值
excerpt: 实现一个可以找到数组内数值均值的函数
---

实现一个函数 `mean(array)`，该函数返回 `array` 内数值的均值（也称为平均值），`array` 是一个数字数组。

## 参数

1. `array` *(Array)*: 数字数组。

## 返回值

*(Number)*: 返回 `array` 中数值的均值。

## 例子

```js
mean([4, 2, 8, 6]); // => 5
mean([1, 2, 3, 4]); // => 2.5
mean([1, 2, 2]); // => 1.6666666666666667
```

如果 `array` 为空，则该函数应返回 `NaN`。

```js
mean([]); // => NaN
```
