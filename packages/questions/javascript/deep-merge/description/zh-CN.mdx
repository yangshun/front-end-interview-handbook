---
title: 深度合并
excerpt: 实现一个将两个对象合并在一起的函数
---

实现一个函数 `deepMerge(objA, objB)`，它接受两个对象并返回一个在深度合并它们之后的新对象：

* 结果对象应包含两个对象的键/值的并集。
* 如果两个对象中都存在相同的键，则合并后的值将来自 `objB`，除非：
  * 两个值都是数组：`objB` 的元素将附加在 `objA` 之后。
  * 两个值都是对象：根据与 `deepMerge` 相同的规则合并对象。
* 合并对象中的数组和对象应该是新实例。

输入对象不应被修改。

## 例子

```js
deepMerge({ a: 1 }, { b: 2 }); // { a: 1, b: 2 }
deepMerge({ a: 1 }, { a: 2 }); // { a: 2 }
deepMerge({ a: 1, b: [2] }, { b: [3, 4] }); // { a: 1, b: [2, 3, 4] }
```
