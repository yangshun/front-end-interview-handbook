---
title: 分块
excerpt: 实现一个函数，该函数将元素数组拆分成指定大小的较小组。
---

实现一个函数 `chunk(array, [size=1])`，它将输入数组分成长度为 `size` 的组，并在**新**数组中返回它们。如果数组无法均匀拆分，则最后一个块将是剩余的元素。该函数不应修改原始输入数组。

## 参数

1. `array` *(Array)*: 要处理的数组。
2. `[size=1]` *(number)*: 每个块的长度。

## 返回值

*(Array)*: 返回新的块数组。

## 例子

```js
chunk(['a', 'b', 'c', 'd']); // => [['a'], ['b'], ['c'], ['d']]
chunk([1, 2, 3, 4], 2); // => [[1, 2], [3, 4]]
chunk([1, 2, 3, 4], 3); // => [[1, 2, 3], [4]]
```

如果 `array` 参数为空，则该函数应返回一个空数组。
